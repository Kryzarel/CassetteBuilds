Doc Leaf is a Plant-type Status Effect Move.

Contents

  • 1 Description
  • 2 Status Effects
  • 3 How to obtain
  • 4 Compatible Species

Description

Gives the target Healing Leaf status for 3 turn(s).

Status Effects

Status EffectKind
Healing Leaf Buff

How to obtain

Doc Leaf may be obtained when a compatible tape levels up.

Doc Leaf can be obtained when a bootleg of one of the following types levels up:

  • Plant
