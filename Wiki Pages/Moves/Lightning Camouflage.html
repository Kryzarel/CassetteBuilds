Lightning Camouflage
Status Effect Move
Type
Lightning
Accuracy100%
Use Cost2 AP
CopyableYes
PriorityNormal (0)

Lightning Camouflage is a Lightning-type Status Effect Move.

Contents

  • 1 Description
  • 2 Status Effects
  • 3 How to obtain
  • 4 Compatible Species

Description

Changes the type of the user. Automatically used at the start of battle for 0 AP.

Status Effects

Status EffectKind
Lightning Coating Transmutation

How to obtain

Lightning Camouflage may be obtained when a compatible tape levels up.

Lightning Camouflage can be obtained when a bootleg of one of the following types levels up:

  • Metal
